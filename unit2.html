The basis of my website, is a border around the whole page, a background image which is different for each page and navbar at the top of each page for the user to be able to go to different pages. 1.5 Describing copyright and other constraints affecting websites
“Copyright protects your work and stops others from using it without your permission”[7]. So basically, if you create something or make something, like a picture, then that picture is copyrighted. The only way for someone else to use that picture is for you to grant permission to them exclusively or have stated somewhere that this picture (or whatever it may be) is available to everyone. When designing my webpage I was aware of copyright, therefore I thought it would be a good idea to use my own photos that I have taken, meaning I don’t have to worry about using other peoples creation.


1.6 Describing access issues that need taking into account - describe issues that people with disabilities may face when using webpages and how we can overcome them.
The three most common tools a web designer should add to their website, is: screen magnifier, screen reader and speech recognition. Whether a person is partially blind, or completely blind, these three tools can help the person use the internet.
